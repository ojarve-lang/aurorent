# Autorendi veebileht

## Mis see projekt on
See on lihtne autorendi veebileht, mis on tehtud PHP ja MySQL abil.

Kasutaja saab:
- vaadata autosid
- otsida autosid
- teha broneeringuid

Admin saab:
- autosid lisada
- autosid muuta
- autosid kustutada
- vaadata broneeringuid

---

## Tehnoloogiad
- PHP
- MySQL
- Bootstrap 5

---

## Kaustad
- public – kasutaja lehed
- admin – admini lehed
- inc – andmebaasi ühendus

---

## Andmebaas
Tabelid:
- cars – autode info
- users – admin kasutaja
- reservations – broneeringud

---

## Funktsioonid
- autode list
- otsing
- detailvaade
- admin login
- broneerimine
- hinna arvutus
- ei luba topelt broneeringut

---

## Kokkuvõte
Õppisin:
- kuidas PHP ja MySQL koos töötavad
- kuidas teha CRUD süsteemi
- kuidas teha login ja sessioon
- kuidas kontrollida andmeid (broneeringud)
